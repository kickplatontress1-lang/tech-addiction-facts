# Факты технологических возможностей

Пример сайта про факты технологических возможностей на Flask.

## Установка Flask

Windows: `pip install flask`

MacOS: `pip3 install flask`

## Запуск локального сервера

> [!CAUTION]
> Если у вас ошибки, не запускайте сервер через ▷! Используйте `python main.py` в терминале!
